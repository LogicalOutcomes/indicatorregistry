Create a logic model or theory of change

We have created a generic logic model for Canadian agencies serving immigrants and refugees. Agencies can select their own indicators and customize their logic model for their specific programs based on the general framework below. Watch the video tutorial for details. You can create your own logic model two ways:

  • You can click on each link below, which will take you to the indicator registry. You can select relevant indicators and export them into pdf or csv.
  • Or, you can sign up for an account and create your own workspace. Then you can select and save indicators within the registry.
