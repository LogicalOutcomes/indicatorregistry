Create a logic model or theory of change

A logic model is a visual tool that shows how different parts of a program interact to produce results. Logic models are like maps or diagrams of a program. They help you see how your initial objectives combine with different sorts of inputs to shape program activities. Activities then produce outputs, which in turn have certain effects on your clients and community.

In evaluation, logic models help us to understand how programs work, and figure out how to measure their outputs and effects. We create a logic model in the earliest stage of an evaluation, when we are developing an evaluation plan.

SIS includes an exciting new tool to help you develop a logic model for each of the programs you want to evaluate. Simply answer the set of questions contained in Create your program logic model and print your results as a pdf. Use these results to edit the OCASI SIS Program Logic Model, and discover all the indicators you need to measure your program's impact.

The OCASI SIS Program Logic Model is built from a free evaluation planning tool developed by Johns Hopkins RADAR program.

We have also created a standard logic model for Canadian agencies serving immigrants and refugees. Agencies can select their own indicators and customize their logic model for their specific programs based on the general framework below.

Follow these steps to create your own logic model:

  • Each link below will take you to the associated indicator in the indicator registry.
  • Select relevant indicators and export them into pdf.
With a SIS subscription, you can store your list of indicators in your workspace. We'll use that list to create custom data collection tools for your agency.
